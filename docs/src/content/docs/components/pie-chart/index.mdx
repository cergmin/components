### Different themes

You can import some themes from the library or create a new one.

Available themes:

- `bluePieChartTheme`
- `greenPieChartTheme`
- `rainbowPieChartTheme`

```javascript
import {
  PieChart,
  bluePieChartTheme,
  greenPieChartTheme,
  rainbowPieChartTheme
} from '@cergmin/components';

const largestCompanies2021 = [
  { title: 'Apple', value: 274 },
  { title: 'Samsung', value: 200 },
  { title: 'Google', value: 182 },
  { title: 'Foxconn', value: 181 },
  { title: 'Microsoft', value: 143 },
  { title: 'Huawei', value: 129 },
  { title: 'Facebook', value: 85 },
];

<PieChart
  theme={bluePieChartTheme}
  data={largestCompanies2021}
/>
<PieChart
  theme={greenPieChartTheme}
  data={largestCompanies2021}
/>
<PieChart
  theme={rainbowPieChartTheme}
  data={largestCompanies2021}
/>
```

### Custom theme

Theme is an array of objects.
Each object describes the appearance of one segment of a pie chart,
so you can create you own theme.

An object that describes the appearance of a segment should have these properties:

- `fill` â€” background of a segment

<Example>
  <PieChart
    theme={[
      { fill: '#002533' },
      { fill: 'rgb(0, 95, 115)' },
      { fill: 'hsl(181deg, 86%, 32%)' },
      { fill: 'orange' },
      { fill: `#bb3e03` },
      { fill: `#ae2012` },
      { fill: `#9b2226` },
    ]}
    data={[
      { title: 'Apple', value: 274 },
      { title: 'Samsung', value: 200 },
      { title: 'Google', value: 182 },
      { title: 'Foxconn', value: 181 },
      { title: 'Microsoft', value: 143 },
      { title: 'Huawei', value: 129 },
      { title: 'Facebook', value: 85 },
    ]}
  />
</Example>

```javascript
import { PieChart } from '@cergmin/components';

const customTheme = [
  { fill: '#002533' },
  { fill: 'rgb(0, 95, 115)' },
  { fill: 'hsl(181deg, 86%, 32%)' },
  { fill: 'orange' },
  { fill: `#bb3e03` },
  { fill: `#ae2012` },
  { fill: `#9b2226` },
];

const largestCompanies2021 = [
  { title: 'Apple', value: 274 },
  { title: 'Samsung', value: 200 },
  { title: 'Google', value: 182 },
  { title: 'Foxconn', value: 181 },
  { title: 'Microsoft', value: 143 },
  { title: 'Huawei', value: 129 },
  { title: 'Facebook', value: 85 },
];

<PieChart theme={customTheme} data={largestCompanies2021} />;
```
