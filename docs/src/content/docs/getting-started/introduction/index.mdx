## Quick start

Install via npm or pnpm:

```bash
npm install @cergmin/components

pnpm add @cergmin/components
```

Import needed component and use it in your app:

```tsx
import { Button } from '@cergmin/components';

const App = () => (
  <>
    <h1>Click this button</h1>
    <Button onClick={() => alert('Hello, world!')}>
        Say hello!
    </Button>
  </>
);
```
